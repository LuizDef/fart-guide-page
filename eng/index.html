Installing Dependencies

  • sox - Command-line audio processing tool.

  • youtube-dl - Used to download and convert videos from youtube (do be careful, do not to commit copyright infringement).

  • libsox-fmt-all - SoX libraries required for playing audio files.

  • ffmpeg - Audio manipulation tool (used for converting videos to mp3 format).


How to install:

Utilize your distribution's package manager to install each of the dependencies.

For Ubuntu, use:

sudo apt install sox libsox-fmt-mp3 youtube-dl ffmpeg


Downloading MP3 from Youtube

We will be downloading this lovely video from Youtube as an example:

Make sure you cd into the folder you wish to download the audio file to.

We'll use the Downloads folder as an example:

cd ~/Downloads

Download and convert to MP3 at the same time with the command:

youtube-dl -x --audio-format mp3 https://youtu.be/Qi1KebO4bzc

Here's a breakdown of this command:

  • youtube-dl - calls youtube-dl to download a video from Youtube.

  • -x - tells youtube-dl to extract the audio from the video.

  • --audio-format mp3 - specifies the format of the audio file when we download it, in this case, MP3.

  • https://youtu.be/Qi1KebO4bzc - URL of the video to be downloaded and converted.



Playing the sound on command-line

Playing a sound file on your command-line is now as simple as typing the command:

play path-to-file.mp3

Where you need to replace the path-to-file.mp3 with the path to your mp3 file.

So in our case:

play ~/Downloads/Reverb\ fart\ sound\ effect-Qi1KebO4bzc.mp3

Will play our super funny fart sound effect lmao.



EXTRA - Make a soundboard with Shortcuts

Many Linux distributions allow you to add command-line commands to keyboard shortcuts.

Using this, we can create our own soundboard combining shortcuts with the play command!

Let's try it on Ubuntu GNOME:

  1. Click on Activities

  2. Type "settings" and open system settings

  3. Go to Keyboard

  4. Scroll down to the bottom to find the + button

  5. You should see this window:


  6. In "Name" you can give your shorcut a name

  7. In "Command" you type your custom command, in our case: play ~/Downloads/Reverb\ fart\ sound\ effect-Qi1KebO4bzc.mp3

  8. Click on "Set Shortcut" and it will ask you to press the shortcut you want to assign, try using this example: Windows Key+Numpad 1

  9. Congratulations, you can now fart by pressing Windows Key+Numpad 1

Note that you can use any command-line command with shortcuts, not just playing sounds. Go wild.
